Detailed Description

Initialize the game creating process. Handles all the basic works, like window loading, and passes the control to allocate the game Subclass this, and proved the actual allocator for the actual game.

The actual purpose of this class is to provide an easy RAII mechanism to hold Game instance. Something as simple as this this: 

int main(int argc, char *argv[])
{
    MyGameConfig gc(GLKVector3Make(0.5, 0.5, 0.5));
    return HideousEngineInit(argc, argv, gc);
